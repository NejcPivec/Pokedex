# Pokedex

Pokedex build with NextJs and Tailwind. It shows only 100 pokemons, but you are able to search between 600 diffrent pokemons.

### Techologies:

- Next.js
- Tailwind (For styling)

### API Calls:

[Pokemon API](https://pokeapi.co/)
